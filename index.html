如何使用jsDelivr镜像服务？

若您需要使用加速服务，只需将 jsDelivr 链接中的 cdn.jsdelivr.net 替换为 jsd.aoe.me即可。比如：

https://cdn.jsdelivr.net/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

替换后变为：

https://jsd.aoe.me/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

若想访问项目文件夹，请使用如下格式：

https://jsd.aoe.me/npm/包名@版本/

示例：

https://jsd.aoe.me/npm/jquery@4.0.0-beta.2/

如何使用Docker镜像服务？

原拉取镜像命令

docker pull library/alpine:latest

      
加速拉取镜像命令

      
docker pull hub.aoe.me/library/alpine:latest

      
修改 /etc/docker/daemon.json 文件

      
{
  "registry-mirrors": [
    "https://hub.aoe.me",
  ]
}

      
重载systemd管理守护进程配置文件

      
sudo systemctl daemon-reload

      
重启 Docker 服务

      
sudo systemctl restart docker

      
一键设置 registry mirror

      
sudo tee /etc/docker/daemon.json
